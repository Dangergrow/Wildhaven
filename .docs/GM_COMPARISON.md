# GM vs Wildhaven — Финальное сравнение и план

## Что в GM есть, у нас нет (критичное)
1. [x] Stability — блоки обрушаются без опоры (ДОБАВЛЕНО)
2. [ ] Key rebinding — все клавиши переназначаются
3. [ ] Separate audio: Music/Effects/Ambient volume
4. [ ] FPS cap: 30/60/Unlimited
5. [ ] Fire spread — огонь перекидывается на соседние блоки
6. [ ] Seasons — влияют на farming, temperature
7. [ ] Animals — hunting, taming, pens, breeding
8. [ ] Trebuchet/catapult — siege weapons

## Что в GM есть, у нас частично
1. [x] Work priorities 1-4 — есть WorkPanel, но не подключено к AI
2. [x] Schedule 24h — есть ColonistSchedule, но не visual grid
3. [x] Roof — есть ColonyServices.AutoRoof, но нет manual placement
4. [x] Medical beds — есть, но нет визуального обозначения
5. [x] Zones — есть ZoneMarker, но нет drag-выделения
