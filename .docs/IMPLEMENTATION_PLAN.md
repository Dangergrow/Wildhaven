  - [ ] Возраст (16-70, слайдер)

  - [ ] Навыки 0-20 (14 навыков, распределение очков)
  - [ ] Перк (+) и Недостаток (-) (баланс)
  - [ ] Предыстория (фермер, солдат, учёный, бандит и т.д.)
- [ ] Кнопка "Случайный колонист" (генерит всё случайно)

**Шаг 3: Выбор места на карте**
- [ ] Глобальная hex-карта с биомами

- [ ] Кнопка "Начать игру"
