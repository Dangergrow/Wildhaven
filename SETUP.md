# Wildhaven — колони-сим с мультиплеером (Unity + AI Agents)

## Что мы делаем

**Wildhaven** — гибрид: графика и строительство в духе Going Medieval + глобальная карта и путешествия как в RimWorld. Своя вселенная, мультиплеер до трёх игроков через Steam (Spacewar).

### Два слоя игры

#### Глобальная карта (RimWorld-style)
- Процедурно генерируемый мир: континенты, океаны, реки, горы, дороги
- Hex-сетка (шестиугольники) для перемещения
- Десятки фракций со своими поселениями, разбросанными по миру
- Каждый гекс имеет: биом, высоту, дороги, реки, ресурсы, населённость
- Ваше поселение отображается на глобальной карте
- Каждый игрок в мультиплеере основывает поселение в своём гексе

#### Локальная карта (Going Medieval-style)
- Воксельный 3D-мир в выбранном гексе: строительство, колонисты, бой
- Именно то, что мы уже описали: слои земли, здания, фермы, производство
- Игроки могут посещать локальные карты друг друга (прибежать караваном)

## Глобальная карта — полная механика

### Генерация мира
- Seed-based процедурная генерация всего мира
- Континенты, океаны, горные хребты, реки, озёра, дороги
- Биомы: тундра, тайга, лес, равнина, болото, пустыня, саванна, тропики
- Размещение фракций: у каждой своя территория (кластер поселений)
- Древние руины, заброшенные лагеря, сокровищницы — разбросаны по миру
- Погода и климат зависят от широты и биома

### Караваны и путешествия
- Формирование каравана: выбираешь колонистов, животных, припасы
- Движение по гексам: скорость зависит от местности, дорог, времени года
- Потребление еды в пути, усталость, риск болезней
- Случайные события в пути: засады, находки, болезни, встреча с торговцем
- Прибытие в точку назначения: торговля, бой, дипломатия, кража
- Можно разбить временный лагерь для отдыха в пути
- Караван можно разделить или объединить

### Фракции (10+)
- Империя, дикари, торговый союз, пираты, культисты, кочевники, рыцари, разбойники, отшельники, мутанты
- Отношения с каждой: -100 (война) до +100 (союз)
- Типы отношений: война, нейтралитет, торговля, союз, вассал
- Действия влияют на отношения: подарки, нападения, освобождение пленных
- Союзники могут присылать помощь при рейдах
- Можно вызывать караван союзников (платно или за очки отношений)

### Поселения NPC
- Каждое поселение фракции — торговая точка
- Ассортимент зависит от типа фракции и её технологического уровня
- Можно напасть на поселение (отдельная боевая карта)
- Можно украсть ресурсы (скрытно)

### Квесты и события на глобальной карте
- Спасти беженца, доставить груз, уничтожить лагерь, построить монумент
- Запросы от фракций: прислать N единиц ресурса к сроку
- Глобальные угрозы: нашествие, чума, голод, засуха
- Древние угрозы: пробуждение механоидов, демонический портал
- Сезонные события: урожай, миграция животных, полярная ночь

### Мультиплеер на глобальной карте
- Каждый игрок основывает своё поселение в любом свободном гексе при старте
- Игроки видят поселения друг друга на глобальной карте
- Караван может дойти до поселения другого игрока (загружается его локальная карта)
- Совместные рейды на NPC-поселения
- Торговля между игроками через караваны
- Общий мир: если один игрок уничтожил фракцию — она исчезла для всех
- PvP: можно напасть на поселение другого игрока (опционально, настройка при старте)

## Локальная карта — полная механика

### Главное меню и настройки
- Главное меню: Новая игра, Загрузить, Мультиплеер, Настройки, Выход
- Настройки: Звук, Графика, Видео, Управление (биндинг клавиш)
- Выбор языка (RUS/ENG)

### Создание колонии
- Создание персонажей (как в Going Medieval): внешность, навыки (0-20), перки/недостатки, история
- Выбор стартовых ресурсов и экипировки
- Выбор фракции (одна из 3 играбельных: поселенцы, кочевники, изгнанники — разные бонусы)
- Выбор места на глобальной карте для основания поселения
- Параметры: размер локальной карты, биом, сложность, частота рейдов, PvP вкл/выкл
- Настройка мультиплеера: пауза разрешена / без паузы / по голосованию

### Генерация локальной карты
- Большая карта: несколько биомов, реки, горы, пещеры
- Воксельный мир: слои земли/камня/руды, вода, перепады высот
- Процедурная генерация с seed
- Терраформинг: копать, засыпать, ровнять

### Колонисты
- Глубокие потребности: голод, жажда, усталость, настроение, комфорт, социализация, рекреация
- Навыки: строительство, добыча, кулинария, интеллект, медицина, бой, ремесло, фермерство, социальный, животноводство, охота, торговля, искусство
- Перки/недостатки: быстрый ученик, трудоголик, ленивый, трус, каннибал, пироман, гурман, аскет, психопат и др.
- История персонажа влияет на навыки и отношения
- Расписание дня: работа, отдых, сон, развлечения
- Отношения: дружба, вражда, любовь, брак
- Боевой дух, ментальные срывы, вдохновение

### Строительство
- Стены, пол, лестницы, крыша, двери, окна, укрепления (разные материалы)
- Зоны: склад, свалка, ферма, комната, госпиталь, тюрьма, храм, мастерская, столовая, кухня
- Многоэтажность, подвалы, башни
- Качество постройки влияет на скорость и настроение
- Разрушение, ремонт, обрушение крыши без опоры
- Blueprint-режим (планировать до ресурсов), копирование/поворот построек
- Двери с правами доступа

### Качество помещений
- Рейтинг: убогая → роскошная
- Влияет на настроение (спальня, столовая) и эффективность (мастерская, лаборатория)
- Зависит от: размера, чистоты, материалов, мебели, искусства, освещения

### Освещение и температура
- Факелы, жаровни, свечи, фонари, люстры
- Темнота = штрафы к работе и бою
- Температура помещений: стены, крыша, источники тепла, сезон
- Тепловые волны/заморозки, одежда по сезону
- Растения требуют света (подземные фермы — только с лампами)

### Ресурсы и крафт
- 50+ типов ресурсов: дерево, камень, металлы, еда, ткани, кожа, драгоценности
- Цепочки производства: дерево→доски→мебель, руда→слитки→оружие
- Верстаки: костёр, верстак, кузница, лаборатория, ткацкий станок, кожевня, пивоварня
- Технологическое древо: 5 эпох
- Заказы на верстаках: «делать пока не X», «бесконечно», копирование настроек
- Приоритеты задач для каждого колониста (1-4)

### Фермерство, охота, животные
- Растениеводство: посадка, полив, сбор, сезонность
- Охота на диких животных, разделка туш
- Приручение и разведение: собаки, козы, куры, лошади, коровы
- Зоны для животных, кормление

### Еда и напитки
- Блюда: сырая → простая → изысканная → пир
- Штраф за однообразие, порча продуктов, погреб
- Алкоголь: пивоварня, винокурня, риск алкоголизма

### Медицина
- Качество перевязки, уровни лекарств
- Болезни: инфекции, чума, дизентерия, грипп
- Операции: ампутация, трепанация
- Чистота госпиталя = залог выздоровления

### Бой и оборона
- Оружие: мечи, луки, копья, топоры, щиты (разное качество)
- Броня: по слотам (голова, тело, ноги)
- Ловушки: шипы, ямы, подъёмный мост, кипящее масло
- Рейдеры: волны, типы врагов
- Осадные орудия у поздних рейдов (требушеты)
- Система тревоги, ранения: переломы, кровотечения, инфекции

### Вода, чистота и погребение
- Колодцы, сбор дождя, орошение
- Грязь, уборка, гигиена
- Трупы: гниение, могилы, кремация

### Захват, пленные, социалка
- Захват рейдеров, тюрьма, вербовка
- Побеги, бунты
- Соц-жизнь: друзья, враги, драки, свадьбы
- Ментальные срывы: запой, ярость, побег

### Развлечения
- Настольные игры, спорт, шахматы, книги, медитация
- Разнообразие обязательно (штраф за однотипный отдых)

### Торговля на локальной карте
- Караваны NPC приходят периодически
- Покупка/продажа, сигнальный костёр
- Репутация влияет на цены

### Время и сохранения
- Перемотка: 1x, 2x, 4x (соло), 1x, 2x, 3x (мультиплеер)
- Пауза: пробел (соло), по настройкам сервера (мультиплеер)
- Автосохранения + ручные, 10+ слотов

### Статистика и QoL
- Богатство колонии, история событий, графики
- Поиск по предметам/колонистам
- Горячие клавиши, группы

## Мультиплеер — полная спецификация

### Создание сервера
- Хост выбирает: глобальный мир (seed), настройки сложности, PvP вкл/выкл, пауза
- Каждый игрок при входе выбирает место для своего поселения на глобальной карте
- Игроки видят поселения друг друга на глобальной карте
- Можно договориться заселиться рядом или далеко

### Совместная игра на одной локальной карте
- Игрок A отправляет караван к поселению игрока B
- По прибытии — игрок A загружается на локальную карту игрока B
- Совместное строительство, охота, бой, крафт
- Деление колонистов: у каждого игрока свои
- Чат, пинги на карте, общие blueprint

### Синхронизация
- Авторитативный сервер (хост), клиенты через Steamworks
- Mirror/FishNet
- Все изменения мира синхронизируются
- Сохранения синхронизированы
- Реконнект при обрыве

### Совместные действия вне поселений
- Совместный рейд на NPC-лагерь (оба игрока на одной боевой карте)
- Совместный караван для дальней экспедиции
- Торговля между игроками через караваны

## Оптимизация
- LOD для вокселей (дальние блоки — упрощённая геометрия)
- Фрустум-куллинг (рендерим только видимое)
- Пул объектов (блоки, предметы, частицы — переиспользуем)
- Чанковая загрузка мира (подгружаем только рядом с камерой)
- Compute Shader для симуляции воды/тепла (на GPU)
- Целевая платформа: Windows 10/11
- Целевой FPS: 60 стабильно на GTX 1060+

## Установка окружения с нуля

### 1. Ollama + модели

```powershell
# Установить Ollama с https://ollama.com/download/windows

# Модель для explore-агента (быстрая, ~5.4 GB, влезает в VRAM)
ollama pull gemma2:9b

# Модель для general + ui-designer (кодовая, 16.9 GB)
ollama run hf.co/DavidAU/Qwen3.6-27B-Heretic-Uncensored-FINETUNE-NEO-CODE-Di-IMatrix-MAX-GGUF:Q4_K_M
```

### 2. Unity

```powershell
# Скачать Unity Hub с https://unity.com/download
# В Unity Hub → Installs → Install Editor → Unity 6 LTS
# При установке выбрать модули: Windows Build Support
```

### 3. Visual Studio Code

```powershell
# Скачать с https://code.visualstudio.com
# Расширения: C#, C# Dev Kit, Unity
```

### 4. Git

```powershell
git clone https://github.com/Dangergrow/Wildhaven.git
```

### 5. Steam + Spacewar (для мультиплеера)

```powershell
# Установить Steam. Затем в браузере:
steam://run/480
# Дождаться установки Spacewar в библиотеке Steam
```

### 6. Открыть проект

```powershell
cd Wildhaven
# Запустить opencode в этой папке
# HANDSHAKE.md и SETUP.md загрузятся автоматически (instructions в opencode.json)
```
