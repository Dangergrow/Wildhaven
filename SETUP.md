# Wildhaven — колони-сим с мультиплеером (Unity + AI Agents)

## Что мы делаем

**Wildhaven** — гибрид: графика и строительство в духе Going Medieval + глобальная карта и путешествия как в RimWorld. Своя вселенная, мультиплеер до трёх игроков через Steam (Spacewar).

### Два слоя игры

**Глобальная карта (RimWorld-style)** — процедурно генерируемый мир на выбор планеты, hex-сетка, десятки фракций, караваны, квесты, путешествия между поселениями игроков.

**Локальная карта (Going Medieval-style)** — воксельный 3D-мир, строительство, колонисты, бой, фермы, подземелья.

## Глобальная карта

### Планеты и генерация мира
- Несколько планет на выбор (разная гравитация, атмосфера, ресурсы)
- Seed-based процедурная генерация: континенты, океаны, горные хребты, реки, озёра, дороги
- Биомы (14): тундра, тайга, лес, равнина, болото, пустыня, саванна, тропики, джунгли, ледяная пустошь, вулканический, грибной лес, кристальные пещеры, мёртвые земли
- Каждый биом: уникальные текстуры земли/камня/растений, своя палитра цветов
- Уникальные ресурсы по биомам: нефть/стекло (пустыня), экзотические фрукты/бамбук (тропики), редкие металлы (горы), кристаллы (ледяные земли)
- Уникальная фауна: скорпионы/верблюды (пустыня), волки/мамонты (тундра), тигры/обезьяны (джунгли), крокодилы/гигантские насекомые (болото)
- Размещение фракций: у каждой своя территория
- Древние руины, заброшенные лагеря, сокровищницы
- Погода и климат зависят от широты, биома и типа планеты

### Караваны и путешествия
- Формирование: выбираешь колонистов, животных, припасы
- Движение по гексам: скорость зависит от местности, дорог, сезона
- Потребление еды, усталость, риск болезней в пути
- Случайные события: засады, находки, болезни, встреча с торговцем
- Прибытие: торговля, бой, дипломатия, кража
- Временный лагерь для отдыха
- Разделение/объединение караванов

### Фракции (10+)
- Империя, дикари, торговый союз, пираты, культисты, кочевники, рыцари, разбойники, отшельники, мутанты
- Отношения: -100 (война) до +100 (союз)
- Типы: война, нейтралитет, торговля, союз, вассал
- Действия влияют на отношения
- Союзники присылают помощь, можно вызывать караван
- NPC-поселения: торговля, нападение, кража

### Квесты и события на глобальной карте
- Спасти беженца, доставить груз, уничтожить лагерь, построить монумент
- Запросы от фракций, глобальные угрозы (нашествие, чума, засуха)
- Древние угрозы: механоиды, демонический портал
- Сезонные события: урожай, миграция, полярная ночь

### Мультиплеер на глобальной карте
- Каждый игрок основывает поселение в своём гексе при старте
- Игроки видят поселения друг друга
- Караван к поселению другого игрока → загружается его локальная карта
- Совместные рейды на NPC, торговля между игроками
- Общий мир: уничтожил фракцию — исчезла для всех
- PvP опционально (настройка при старте)

## Локальная карта

### Главное меню и настройки
- Новая игра, Загрузить, Мультиплеер, Настройки, Выход
- Настройки: Звук, Графика, Видео, Управление (биндинг клавиш)
- Выбор языка (RUS/ENG)

### Создание колонии
- Создание персонажей: внешность, навыки (0-20), перки/недостатки, история
- Выбор стартовых ресурсов и экипировки
- Выбор фракции (3 играбельных: поселенцы, кочевники, изгнанники)
- Выбор места на глобальной карте для поселения
- Параметры: размер карты, биом, сложность, рейды, PvP
- Настройка мультиплеера: пауза / без паузы / по голосованию

### Генерация локальной карты
- Большая карта, несколько биомов, реки, горы, пещеры
- Воксельный мир: слои земли/камня/руды, вода, перепады высот
- Seed-based генерация, терраформинг

### Колонисты
- Потребности: голод, жажда, усталость, настроение, комфорт, социализация, рекреация
- Навыки (13): строительство, добыча, кулинария, интеллект, медицина, бой, ремесло, фермерство, социальный, животноводство, охота, торговля, искусство
- Перки/недостатки: быстрый ученик, трудоголик, ленивый, трус, каннибал, пироман, гурман, аскет, психопат и др.
- История персонажа, расписание дня, отношения, ментальные срывы

### Анимации персонажей
- Ходьба/бег (с инструментом и без, по лестницам), idle (несколько вариантов)
- Работа: копает, рубит, строит, пилит, сажает, собирает, готовит, куёт, шьёт, лечит, хоронит
- Переноска: ящик, мешок, тело, тачка
- Бой: меч, топор, копьё, лук, блок, уклонение, урон, смерть
- Социалка: разговор, драка, объятия, танец
- Еда/сон: за столом, пьёт, в кровати, на полу
- Эмоции: радость, грусть, ярость, страх
- Особые: ранен (хромает), пьян (шатается), болен (кашляет), срыв (по типу)
- IK: поворот корпуса, поворот головы, инструмент в руках (меняется визуально)
- Источник анимаций: Mixamo (бесплатно) + Unity Asset Store (средневековые наборы)
- Агенты пишут: AnimatorController, Blend Trees, IK, переходы, смену инструмента

### Строительство и подземелья
- Стены, пол, лестницы, крыша, двери, окна, укрепления (разные материалы)
- Зоны: склад, свалка, ферма, комната, госпиталь, тюрьма, храм, мастерская, столовая, кухня
- Многоэтажность: подвалы, наземные этажи, башни
- Копать вглубь: земля → камень → руда → глубокая порода
- Подземные базы: комнаты, склады, фермы (с лампами), мастерские
- Шахты: добыча из стен, расширение помещений
- Укрепления под землёй: бункеры, туннели, подземные ловушки
- Подземные реки и озёра (опасность затопления)
- Своды и опоры: без колонн обрушится
- Blueprint, копирование/поворот, двери с правами
- Качество постройки, ремонт

## Управление (Going Medieval-style, полное)

### Камера
- WASD / стрелки — движение
- Q / E — поворот (45° шаг)
- Колёсико мыши — зум
- Средняя кнопка мыши (зажать) — свободный поворот
- ПКМ (по земле) — снять выделение
- Home — сброс камеры
- Double-click — центрировать на объект
- Рамка (drag ЛКМ) — выделить несколько, Shift добавить, Ctrl исключить

### Этажность
- Tab / Shift+Tab — переключение этажа
- Ползунок этажей в UI, прозрачность стен
- Кнопка скрытия крыш

### Режимы
- Tab — Architect/Work/Zone/Orders
- F1-F4 — быстрый доступ
- 1-9 — подкатегории
- Backspace — отмена, Ctrl+Z — undo
- R — поворот, G — перетащить

### Время
- Пробел — пауза
- Num1/Num2/Num3 — скорость

### Колонисты
- ЛКМ — выбрать, ПКМ — приказ идти/атаковать/подобрать
- F — приоритетный приказ, R — боевой режим, C — отмена

### Строительство
- ЛКМ — разместить, ПКМ — отмена инструмента
- Shift+ЛКМ — несколько, Delete — снести
- Drag ЛКМ — линия/область

### Глобальная карта
- ЛКМ по каравану — выбрать, ПКМ — отправить
- Удержание ПКМ — маршрут, M — переключение карт

### Меню
- Esc — меню, F5 — сохранение, F9 — загрузка
- F10 — скриншот, F11 — скрыть UI
- I — инвентарь, B — строительство, N — зоны
- J — задания, T — исследования, H — история
- Enter — чат (мультиплеер)

### Настройка управления
- Все клавиши переназначаются (Настройки → Управление)
- Мышиные кнопки переназначаются
- Геймпад: базовая поддержка (опционально)
- Чувствительность, инверсия, профили
- Сброс на дефолт

## Требования к коду (ОБЯЗАТЕЛЬНО)

### Комментарии
- Каждый .cs файл — в начале что за класс
- Каждый public метод — `<summary>` (описание, параметры, возврат)
- Каждое публичное поле — комментарий (что хранит, единицы измерения)
- Сложная логика — построчные комментарии
- Magic numbers запрещены (только const с комментарием)
- TODO / FIXME / HACK с пояснением

### Стиль
- PascalCase классы/методы, camelCase переменные, _camelCase приватные
- Один класс = один файл, не больше 300 строк
- #region для группировки
- Удалять неиспользуемые using

### Архитектура
- Синглтоны через DI, не FindObjectOfType в Update
- ScriptableObject для всех конфигов (предметы, рецепты, фракции)
- Mirror: [SyncVar], [Command], [ClientRpc] для сети
- Префабы в Resources/Prefabs

## Прогнозируемые проблемы (учтены в архитектуре)

### Совместимость сохранений
- Версионирование .sav файлов, цепочка миграций
- Устаревшие поля через [Obsolete] с дефолтами

### Десинхронизация в мультиплеере
- Авторитативный сервер (все решения — хост)
- Хеш-сумма мира раз в N секунд
- Таймстемпы у событий

### Производительность
- Чанки 16x16x16, рендер только видимых
- AI волнами, LOD, Object Pool
- Кеширование путей, инкрементальный патфайндинг

### Память
- Чанки загружаются/выгружаются
- Текстурные атласы, Sprite Atlas, Addressables
- Лимит предметов на земле

### Анимации
- Слои: Base, Upper Body, Face
- IK для рук/ног, Blend Tree, аватар-маски

### UI
- Canvas — Scale With Screen Size
- Тест на 1366x768, 1920x1080, 2560x1440, 4K

### Steamworks
- Абстракция ISteamService — легкая замена Spacewar на свой AppID

### Детерминированная генерация
- System.Random(seed), одинаковый порядок на всех машинах
- Тест: два ПК с одним seed — одинаковый хеш мира

### Конфликты в мультиплеере
- Система «заявок» на блоки (кто первый)
- Склад: общий или раздельный (настройка)

### Моддинг
- XML/JSON конфиги для новых предметов, рецептов, фракций
- Harmony patching для глубоких правок
- Всё через ScriptableObject
