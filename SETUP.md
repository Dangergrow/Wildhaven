# Wildhaven — колони-сим с мультиплеером (Unity + AI Agents)

## Что мы делаем

**Wildhaven** — игра в духе Going Medieval, но со своей вселенной, расширенными механиками и мультиплеером до трёх игроков через Steam (Spacewar).

### Полный список систем и механик

#### Главное меню и настройки
- Главное меню: Новая игра, Загрузить, Мультиплеер, Настройки, Выход
- Настройки: Звук (громкость музыки/эффектов), Графика (разрешение, качество, полноэкранный режим), Видео (VSync, FPS cap), Управление (биндинг клавиш, чувствительность)
- Выбор языка (RUS/ENG)

#### Создание колонии
- Создание персонажей (как в Going Medieval): выбор внешности, распределение навыков, perks/недостатки, история
- Выбор стартовых ресурсов и экипировки
- Выбор параметров мира: размер карты (очень большой), биом, сложность, частота рейдов, настройки мультиплеера

#### Генерация и мир
- ОЧЕНЬ большая карта: несколько биомов, реки, горы, пещеры, ресурсы по зонам
- Воксельный мир: слои земли/камня/руды, вода, перепады высот
- Процедурная генерация с seed (одинаковый мир для всех игроков)
- Терраформинг: копать, засыпать, ровнять

#### Колонисты
- Глубокие потребности: голод, жажда, усталость, настроение, комфорт, социализация, религия, рекреация
- Навыки (как в Going Medieval): строительство, добыча, кулинария, интеллект, медицина, бой, ремесло, фермерство и др. — каждый от 0 до 20
- Perks/недостатки: быстрый ученик, трудоголик, ленивый, трус, каннибал и т.д.
- История персонажа (влияет на навыки и отношения)
- Расписание дня (работа/отдых/сон/развлечения)
- Отношения между колонистами: дружба, вражда, любовь, брак
- Боевой дух, ментальные срывы, вдохновение

#### Строительство
- Все типы построек: стены (дерево/камень/кирпич/металл), пол, лестницы, крыша, двери, окна, укрепления
- Зоны: склад, свалка, growing zone, комната (спальня, столовая, кухня, мастерская, храм, госпиталь, тюрьма)
- Многоэтажность: подвалы, первые этажи, башни
- Качество постройки влияет на скорость работы и настроение
- Разрушение и ремонт

#### Ресурсы и крафт
- 50+ типов ресурсов: дерево, камень, металлы, еда, ткани, кожа, химикаты, драгоценности
- Цепочки производства: дерево→доски→мебель, руда→слитки→оружие/инструменты
- Верстаки: костёр, верстак, кузница, лаборатория, алхимический стол, ткацкий станок, кожевенная мастерская
- Технологическое древо: 5 эпох, каждая открывает новые рецепты

#### Фермерство и охота
- Растениеводство: посадка, полив, сбор урожая, сезонность
- Охота: дикие животные, разделка туш
- Приручение и разведение животных: собаки, козы, куры, лошади

#### Бой и оборона
- Оружие: мечи, луки, арбалеты, копья, топоры, щиты (разное качество)
- Броня: кожаная, кольчужная, латная (по слотам: голова, тело, ноги)
- Ловушки: шипы, ямы, подъёмный мост, кипящее масло
- Рейдеры: волны с растущей сложностью, разные типы врагов
- Система тревоги: колонисты автоматически хватают оружие
- Ранения: переломы, кровотечения, инфекции — требуют лечения

#### Мультиплеер (до 3 игроков)
- Авторитативный сервер (хост) + клиенты
- Деление колонистов между игроками
- Совместное строительство, бой, крафт
- Лобби через Steamworks (Spacewar AppID)
- Приглашение друзей через Steam
- Чат
- Пауза: при создании сервера выбор — «пауза разрешена» (любой игрок ставит) / «без паузы» (только темп игры: 1x, 2x, 3x) / «пауза по голосованию»
- Реконнект при обрыве соединения
- Сохранения синхронизируются между игроками

#### Время и сохранения
- Перемотка времени: 1x, 2x, 4x (синглплеер), 1x, 2x, 3x (мультиплеер без паузы)
- Пауза (пробел) в синглплеере и мультиплеере с разрешённой паузой
- Автосохранения: настраиваемый интервал (каждые 5/10/15/30 минут, выключено)
- Ручные сохранения: F5 — быстрое, меню — с именем
- Слоты сохранений: 10+ ручных + 5 авто

#### Собственная вселенная
- 3 уникальные фракции со своей эстетикой, бонусами и стартовыми условиями
- Биомы: лес, пустыня, горы, болото, тайга — разная сложность
- События: чума, наводнение, засуха, торговец, паломник, странник
- Дипломатия: торговля, союз, война с фракциями

#### Оптимизация
- LOD-система для вокселей (дальние блоки — упрощённая геометрия)
- Фрустум-куллинг (рендерим только видимое)
- Пул объектов (блоки, предметы, частицы — переиспользуем)
- Chunk-based загрузка мира (подгружаем только рядом с камерой)
- Compute Shader для симуляции воды/тепла (на GPU)
- Целевая платформа: Windows 10/11
- Целевой FPS: 60 стабильно на GTX 1060+

### Архитектура разработки

- **Основной ИИ (DeepSeek V4 Pro)** — стратегия, архитектура, распределение задач между агентами, контроль качества
- **general-агент (NEO-CODE 27B)** — пишет код, реализует механики
- **ui-designer (NEO-CODE 27B)** — интерфейсы, HUD, меню
- **explore (gemma2:9b)** — поиск по кодовой базе, анализ

## Установка окружения с нуля

### 1. Ollama + модели

```powershell
# Установить Ollama с https://ollama.com/download/windows

# Модель для explore-агента (быстрая, ~5.4 GB, влезает в VRAM)
ollama pull gemma2:9b

# Модель для general + ui-designer (кодовая, 16.9 GB, ~70% на GPU)
ollama run hf.co/DavidAU/Qwen3.6-27B-Heretic-Uncensored-FINETUNE-NEO-CODE-Di-IMatrix-MAX-GGUF:Q4_K_M
```

### 2. Unity

```powershell
# Скачать Unity Hub с https://unity.com/download
# В Unity Hub → Installs → Install Editor → Unity 6 LTS
# При установке выбрать модули: Windows Build Support, WebGL (опционально)
```

### 3. Visual Studio Code

```powershell
# Скачать с https://code.visualstudio.com
# Расширения: C#, C# Dev Kit, Unity
```

### 4. Git

```powershell
git clone https://github.com/Dangergrow/Wildhaven.git
```

### 5. Steam + Spacewar (для мультиплеера)

```powershell
# Установить Steam. Затем в браузере:
steam://run/480
# Дождаться установки Spacewar в библиотеке Steam
```

### 6. Открыть проект

```powershell
cd Wildhaven
# Запустить opencode в этой папке
# HANDSHAKE.md загрузится автоматически (прописан в opencode.json → instructions)
# При первом запуске также прочитает SETUP.md для понимания проекта
```

## Агенты

Конфиг в `.opencode/agents/`. Агенты загружаются автоматически при запуске opencode в папке проекта.

| Агент | Модель | Назначение |
|---|---|---|
| explore | gemma2:9b | Поиск по коду, анализ файлов |
| general | NEO-CODE 27B | Написание кода, логика, архитектура |
| ui-designer | NEO-CODE 27B | UI/UX, верстка, интерфейсы |

## Технологический стек

- **Движок**: Unity 6 LTS (C#)
- **Мультиплеер**: Mirror + Steamworks (Spacewar AppID)
- **Графика**: Voxel-движок, стилизация low-poly
- **Ассеты**: Kenney.nl, Quaternius.com (бесплатно)
- **Контроль версий**: Git + GitHub
