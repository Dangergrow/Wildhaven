# Wildhaven — Руководство игрока

## Управление

| Клавиша | Действие |
|---|---|
| WASD / Стрелки | Движение камеры |
| Q / E | Поворот камеры на 45° |
| Колёсико мыши | Приблизить/отдалить |
| Средняя кнопка мыши (зажать) | Свободный поворот |
| Home | Сброс камеры в центр |
| Пробел | Пауза |
| Num1 / Num2 / Num3 | Скорость времени 1x / 2x / 4x |

## Режимы (нижняя панель)

| Клавиша | Режим |
|---|---|
| F1 | Architect (строительство) |
| F2 | Work (приоритеты работы) |
| F3 | Zone (зоны) |
| F4 | Orders (приказы) |
| Tab | Переключение между режимами |

## Строительство (F1 — Architect)

В панели справа — 6 категорий блоков: Walls, Floors, Furniture, Production, Defense, Misc.
Кликни категорию, потом кликни блок.

| Действие | Клавиши |
|---|---|
| Поставить блок | ЛКМ по поверхности |
| Сломать блок | ПКМ по блоку |
| Линия блоков | Зажать ЛКМ и тянуть |
| Область 3×3 сноса | Shift + ПКМ |
| Blueprint (построят позже) | Shift + ЛКМ |
| Копировать структуру | Ctrl + C |
| Вставить структуру | Ctrl + V |

## Колонисты

### Выбор и приказы
| Действие | Клавиши |
|---|---|
| Перейти в режим выбора | B |
| Выбрать колониста | ЛКМ по колонисту |
| Приказ идти | ПКМ по земле |
| Приказ копать | ПКМ по блоку |
| Приказ атаковать | ПКМ по врагу |
| Снять выделение | Backspace |

### Панель колониста (при выделении)
5 вкладок:
- **Skills** — навыки 0-20, перк, недостаток
- **Gear** — оружие, броня, инструмент
- **Items** — инвентарь
- **Health** — здоровье, настроение, потребности
- **Sched** — расписание 24ч

### Draft (боевой режим)
Нажми **R** когда колонист выделен — переключит боевой режим.

## Сохранения

| Клавиша | Действие |
|---|---|
| F5 | Сохранить игру |
| F9 | Загрузить игру |

Сохранения автоматические при выходе. При старте если есть сейв — загрузится.

## Настройки

Нажми **Esc** — меню настроек: громкость музыки/эффектов, качество графики, полный экран.

## Создание персонажа

При новой игре:
1. Выбери одного из 3 колонистов (вкладки сверху)
2. Настрой имя (кнопка Name), возраст (Age -/+)
3. Распредели очки навыков (30 очков, кнопки +/-)
4. Выбери Перк и Недостаток (баланс)
5. Выбери Предысторию
6. Цвет волос и телосложение
7. Нажми Random для случайного колониста
8. Нажми START GAME

## Торговля

Караваны приходят случайно. Когда появится панель торговли:
- **Buy Selected** — купить первый товар в списке
- **Sell All** — продать все не-денежные предметы
- **Close** — закрыть

Цены зависят от репутации с фракциями.

## Ресурсы

В левом верхнем углу: W(дерево) S(камень) F(еда) M(металл)
Собираются в инвентаре колонистов. Пополняются от добычи, сбора, охоты.

## События и квесты

Квесты появляются автоматически (до 3 активных). Типы:
- **Rescue** — спасти беженца
- **Deliver** — доставить припасы
- **Destroy** — уничтожить лагерь
- **Build** — построить монумент

Награда в копейках. У квестов есть таймер — не просрочь!

## Бой

Колонисты автоматически атакуют врагов в радиусе.
- С луком/арбалетом — стреляют с дистанции
- Без оружия — ближний бой
- R — принудительный боевой режим
- ПКМ по врагу — приказ атаки

Враги приходят волнами (рейды). Сложность растёт с богатством.

## Системы жизнеобеспечения

- **Еда** — готовится из рецептов (лагерный костёр, плита, печь, пивоварня)
- **Медицина** — лечит болезни и раны. Нужны медицинские койки.
- **Температура** — зависит от сезона, стен, глубины. Влияет на порчу еды.
- **Освещение** — темнота снижает настроение. Лампы от электричества.
- **Красота** — мрамор, стекло, золото улучшают настроение.

## Электричество

- **Генератор** — блоки IronOre/CopperOre/GoldOre = провода
- **Лампа** — блоки Glass рядом с проводами светятся
- Сеть считается автоматически (flood-fill)

## Религия

- **Алтарь** — блок Marble = молитва
- **Жертвенник** — блок Obsidian
- Ритуалы дают бонус к настроению всей колонии

## Стабильность

Блоки без опоры обрушаются. Строй колонны и фундамент.
Многоэтажные здания требуют стен на нижних этажах.

## Сезоны

4 сезона по ~15 дней:
- **Весна** — норма
- **Лето** — тепло, фермы быстрее
- **Осень** — норма
- **Зима** — холодно, фермы медленнее

## Семья

Колонисты женятся при отношении 80+. Могут родиться дети.
Дети растут в 3 раза быстрее. До 14 лет не работают.

## Животные

12 видов на карте. Охота даёт мясо.
С навыком Animal Handling можно приручить.
Приручённые размножаются.

## Пленные

Врагов можно брать в плен. В тюрьме теряют сопротивление.
С социальным навыком можно завербовать.
Либо казнить.
